# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in this project, please report it
privately via [GitHub Security Advisories](https://github.com/almondoo/review-agent/security/advisories/new).

Please do **not** report security issues through public channels (forks,
discussions, or social media) until they have been addressed.

You can expect an initial response within a reasonable timeframe, though as
this is a personal project, response times are best-effort.
