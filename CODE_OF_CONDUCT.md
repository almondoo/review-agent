# Code of Conduct

## Scope

This Code of Conduct applies to behavior in this project's GitHub Issues and
Discussions. It does **not** govern a contribution workflow — this repository
is published for reference and does not accept external contributions. See
[CONTRIBUTING.md](./.github/CONTRIBUTING.md) for the project stance on
pull requests and issue reports.

## Our Standard

Interaction in this space should be respectful and constructive. The following
behaviors are expected:

- Using welcoming and professional language.
- Being respectful of different viewpoints and experiences.
- Focusing on technical accuracy rather than personal criticism.

The following behaviors are not acceptable:

- Harassment, insults, or derogatory comments of any kind.
- Personal or political attacks.
- Publishing others' private information without explicit permission.
- Any other conduct that a reasonable person would consider inappropriate.

## Enforcement

Instances of unacceptable behavior may be reported to the project maintainer
at **tsubasa.engineer@gmail.com**. The maintainer will review all reports
and respond as deemed appropriate. Responses may include removing content or
blocking participants.
